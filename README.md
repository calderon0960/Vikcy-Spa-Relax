# Vicky Spa Relax 🧖‍♀️✨

## 📝 Descripción General
Sitio web oficial de **Vicky Spa Relax**, negocio de bienestar, estética y terapias ubicado en Cabo San Lucas, Baja California Sur.

El objetivo es mostrar todos los servicios, facilitar la reserva directa por WhatsApp, presentar la galería de trabajos y compartir datos de contacto y redes sociales. El diseño es elegante, adaptado a todos los dispositivos y **no incluye precios ni pagos en línea**.
